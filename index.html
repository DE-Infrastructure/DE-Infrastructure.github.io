🌐 HTTP Сервисы

Airflow

Provider: docker

Платформа для создания, планирования и мониторинга рабочих процессов (workflows). Управляйте DAG'ами и автоматизируйте ETL-процессы.

Открыть Airflow

Gitea

Provider: docker

Лёгкий и быстрый сервис управления Git-репозиториями. Альтернатива GitHub для внутренней разработки с поддержкой CI/CD.

Открыть Gitea

Jupyter

Provider: docker

Интерактивная среда для анализа данных, машинного обучения и научных вычислений. Поддержка Python, R, Julia и других языков.

Открыть Jupyter

Kafka UI

Provider: docker

Веб-интерфейс для управления Apache Kafka. Просмотр топиков, consumer groups, мониторинг потоков данных.

Открыть Kafka UI

MinIO

Provider: docker

Высокопроизводительное объектное хранилище с совместимостью Amazon S3 API. Храните и управляйте большими объёмами данных.

Открыть MinIO

OpenProject

Provider: docker

Платформа для управления проектами с поддержкой Agile, Scrum и Kanban. Планируйте задачи, треккайте прогресс и организуйте команду.

Открыть OpenProject

Spark Master

Provider: docker

Главный узел Apache Spark кластера. Управление распределёнными вычислениями для обработки больших данных.

Открыть Spark Master

Spark Worker

Provider: docker

Рабочий узел Apache Spark. Выполняет задачи распределённых вычислений под управлением Spark Master.

🔌 Иные подключения (TCP/SSH)

PostgreSQL

Provider: docker

Объектно-реляционная СУБД с поддержкой сложных запросов, транзакций и расширений. Надёжное решение для OLTP.

Хост: postgres.de-infra.servehttp.com
Порт: 9002
База данных: mydb
База данных: postgres

Greenplum

Provider: docker

Массивно-параллельная СУБД на базе PostgreSQL. Для аналитики и хранилищ данных (Data Warehouse).

Хост: greenplum.de-infra.servehttp.com
Порт: 9003
Протокол: TCP (PostgreSQL)
Пользователь: gpadmin

ClickHouse

Provider: docker

Колоночная СУБД для онлайн аналитической обработки (OLAP). Высокая производительность на больших объёмах данных.

Хост: clickhouse.de-infra.servehttp.com
Порт: 9004
Протокол: TCP (native)
HTTP порт: 8123

Gitea-SSH

Provider: docker

Лёгкий и быстрый сервис управления Git-репозиториями. Альтернатива GitHub для внутренней разработки с поддержкой CI/CD.

Хост: gitea.de-infra.servehttp.com
Порт: 9010

Организация: de-infrastructure
